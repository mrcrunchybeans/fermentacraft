# FermentaCraft

**FermentaCraft** is a fermentation tracking app for homebrewers and craft fermenters — track recipes, batches, gravity readings, iSpindel devices, and more.

## Downloads

The latest Windows release (EXE + MSIX) is available on the [Releases](https://github.com/mrcrunchybeans/fermentacraft/releases) page.

| Platform | Distribution |
|----------|-------------|
| Windows  | [GitHub Releases](https://github.com/mrcrunchybeans/fermentacraft/releases) · Microsoft Store |
| iOS      | App Store |
| Android  | Google Play |

## Source Code

FermentaCraft is closed-source. This repository is used for distributing Windows release artifacts only.
